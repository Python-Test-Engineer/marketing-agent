# SEO and Content Marketing

SEO is the engine that gives content long-term compounding value. Every piece of content in your [[Content Calendar Planning]] should have an organic discovery angle.

## Pillar + Cluster Model

```
Pillar Page (broad topic, 2,000–4,000 words)
    ├── Cluster Post 1 (specific subtopic)
    ├── Cluster Post 2
    ├── Cluster Post 3
    └── ...all internally linked back to Pillar
```

This signals topical authority to search engines and mirrors how [[Audience Research and Personas|audiences]] naturally navigate information.

## Keyword Research Process

1. **Seed keywords** — Core terms from your [[Brand Voice and Messaging|messaging pillars]]
2. **Expand** — Use Ahrefs, Semrush, or Google Search Console
3. **Qualify by:**
   - Search volume (>500/mo for cluster, >2k for pillar)
   - Keyword difficulty (KD <40 for new sites)
   - Intent (informational → TOFU, commercial → MOFU/BOFU)
4. **Map to content brief** → [[Content Calendar Planning]]

## On-Page SEO Essentials

| Element | Best Practice |
|---------|--------------|
| Title tag | Primary keyword near front, <60 chars |
| Meta description | Include keyword, action verb, <155 chars |
| H1 | Exact or close match to title tag |
| URL slug | Short, keyword-rich, hyphens only |
| Internal links | 3–5 per post, descriptive anchor text |
| Image alt text | Descriptive, includes keyword naturally |
| Content length | Match or exceed top 3 SERP competitors |

## Content for Search Intent Types

| Intent | Content Format | Funnel Stage |
|--------|---------------|--------------|
| Informational | How-to guides, explainers | TOFU |
| Navigational | Brand/product pages | Any |
| Commercial | Comparisons, reviews, listicles | MOFU |
| Transactional | Landing pages, pricing pages | BOFU |

## Technical SEO Quick Checklist

- [ ] Site speed: Core Web Vitals passing
- [ ] Mobile-first indexed
- [ ] XML sitemap submitted
- [ ] Canonical tags set correctly
- [ ] Schema markup (Article, FAQ, HowTo)
- [ ] No broken internal links

## Measuring SEO Performance

Track in [[Content Metrics and Analytics]]:
- Organic sessions by page
- Keyword rankings (tracked keywords)
- Click-through rate (CTR) from Search Console
- Pages per session from organic
